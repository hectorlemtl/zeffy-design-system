# Badge

Inline pill used for status chips, eyebrow labels, and feature highlights.

## Variants
- `brand` (default) — bg `color.brand.primary-moderate` (#e0e0ff), text `color.brand.primary-dark`.
- `success` — bg `#bae5de`, text `color.status.success`.
- `warning` — bg `#fff4d6`, text `color.status.warning`.
- `danger`  — bg `color.status.danger-lightest`, text `color.status.danger-dark`.
- `neutral` — bg `color.surface.muted`, text `color.text.secondary`.

## Tokens
- Radius: `var(--radius-round)` (pill).
- Padding: `4px 10px`.
- Font: `font-size.xs` (12px), `font-weight.medium` (500), `letter-spacing.wide`.
- Text transform: none (Zeffy sparingly uses ALL CAPS — reserved for `text-style-allcaps` column labels).
