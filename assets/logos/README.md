# Logos

| File                   | Purpose                          |
|------------------------|----------------------------------|
| `zeffy-logo.svg`       | Primary wordmark, navbar / footer |
| `favicon.png`          | 32×32 favicon                    |
| `apple-touch-icon.png` | iOS home-screen icon             |

**Ownership.** All files are © Zeffy and are the canonical brand marks served from the production marketing site (`cdn.prod.website-files.com` / Webflow CDN for `www.zeffy.com`). Do not alter the proportions or recolor the wordmark.

**Clear space.** Keep at least the height of the "z" glyph as clear space on all sides.

**Monochrome use.** When a monochrome version is required (print, watermark), export the wordmark filled with `--color-brand-primary-darkest` (#06006d) or pure white.
